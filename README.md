# cross-stream-examples-asciiquarium-rs

A minimal [cross.stream](https://cross.stream) example site: one
[http-nu](https://github.com/cablehead/http-nu) handler at `serve.nu` in the repo root.

Deploy it through the customerenv admin app — pick a label, and it goes live at
`<label>.<tenant>.cross.stream`.

Basic on purpose: the starting point an asciiquarium-rs site grows from.
